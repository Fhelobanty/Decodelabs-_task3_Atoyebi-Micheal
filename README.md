# Decodelabs DevOps Project 3 - CI/CD Pipeline Basics

## Project Overview

This project demonstrates the fundamentals of Continuous Integration (CI) using GitHub Actions.

## Technologies Used

- Git
- GitHub
- GitHub Actions
- HTML

## Pipeline Workflow

The pipeline automatically:

- Checks out repository code
- Verifies index.html exists
- Displays validation messages
- Lists repository files

## Trigger

The workflow runs automatically whenever code is pushed to the main branch.

## Learning Outcomes

- Understanding CI/CD concepts
- Creating automated workflows
- Using GitHub Actions
- Working with pipeline stages
